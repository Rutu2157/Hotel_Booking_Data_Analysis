# 📊 Hotel Booking Data Analysis

## 📌 Overview :

This project focuses on analyzing hotel booking data to understand customer behavior, booking patterns, and cancellation trends. The analysis is performed using Python, covering data loading, cleaning, and exploratory data analysis (EDA) to generate meaningful business insights.

## 📂 Dataset :

The dataset contains hotel booking records with key features such as:
  - Hotel type (City Hotel / Resort Hotel)
  - Booking dates and stay duration
  - Booking cancellation status
  - Number of guests
  - Customer country
  - Pricing and reservation details
  
The dataset required preprocessing due to missing values and inconsistent data.

## 🛠️ Tools & Technologies :
  - Python (Pandas, NumPy)
  - Jupyter Notebook
  - Matplotlib / Seaborn
    
## ⚙️ Project Steps :
1. Data Loading
  - Imported dataset using Pandas
  - Explored structure using .info() and .head()
    
2. Data Cleaning
  - Handled missing values
  - Converted date columns into proper format
  - Standardized column names
  - Removed irrelevant or duplicate data
    
3. Exploratory Data Analysis (EDA)
  - Analyzed booking trends over time
  - Studied cancellation patterns
  - Compared City Hotel vs Resort Hotel bookings
  - Analyzed customer distribution by country
  - Examined pricing and stay duration
    
## 📊 Dashboard :
  - Created visualizations using Python libraries
  - Insights presented through charts (bar, line, pie)
  - Helps in understanding booking and cancellation behavior
    
## 📈 Results :
  - High cancellation rates observed
  - City hotels received more bookings than resort hotels
  - Seasonal trends identified in bookings
  - Majority of bookings came from specific countries
  - Pricing influenced booking and cancellation behavior

## ✅ Conclusion :

This project demonstrates a structured approach to analyzing hotel booking data using Python. It highlights skills in data cleaning, exploratory data analysis, and visualization, while extracting actionable insights that can help improve booking strategies and reduce cancellations.
